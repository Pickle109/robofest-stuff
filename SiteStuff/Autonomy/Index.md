## Autonomy

to move [automonously](https://dictionary.cambridge.org/us/dictionary/english/autonomously) the robot uses the line and color sensors .

### Movement  
for movement the robot uses [Lego Spike medium motors](https://education.lego.com/en-us/products/lego-technic-medium-angular-motor/45603/)

### Line following  

for line following we use [Lego Spike prime](https://spike.legoeducation.com/prime/lobby/) and we are using the hub and the color sensors to stay on the line
